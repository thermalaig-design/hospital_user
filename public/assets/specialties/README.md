Drop custom PNG/SVG icons for each specialty here.

Filenames should match the `id` field in `SPECIALTIES` (e.g. `medicine.png`, `cardiology.svg`, etc.).
The app will automatically look for these under `/assets/specialties/<id>.<ext>` and display them in place of the emoji.

If no file is present, the code falls back to the emoji defined in `Appointments.jsx`.

You can copy the images shown in the screenshot (Android emoji-style) into this folder if you prefer them as files.

Recommended size: 64×64 or 128×128px with transparent background.
